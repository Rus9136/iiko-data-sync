Инструкция

Формат JSON файла должен быть массивом объектов продуктов:

[
  {
    "id": "uuid",
    "name": "Название",
    "code": "код",
    "deleted": false,
    ...
  }
]

Поддерживаемые поля:

  • id (UUID)
  • name
  • code
  • num
  • deleted
  • description
  • parent
  • category
  • modifiers
